Hướng dẫn cấu hình

File cấu hình

Tất cả file cấu hình được lưu trong thư mục configs/. Các file cấu hình chính gồm:

config.json Bắt buộc

File cấu hình chính với API Key, cổng, cài đặt nhà cung cấp model (Tự động tạo sau khi lưu quản lý cấu hình)

provider_pools.json Bắt buộc

Cấu hình provider pool cho luân phiên đa tài khoản và chuyển dự phòng (Tự động tạo sau khi lưu node)

plugins.json Tùy chọn

Cấu hình plugin để bật/tắt plugin hệ thống

pwd Tùy chọn

File mật khẩu quản trị, mật khẩu mặc định là admin123

Chi tiết cấu hình chính (config.json)

Cài đặt cơ bản

Tham số Loại Mặc định Mô tả
REQUIRED_API_KEY string - API Key bắt buộc để truy cập dịch vụ này
SERVER_PORT number 3000 Cổng lắng nghe dịch vụ
HOST string 0.0.0.0 Địa chỉ lắng nghe dịch vụ
MODEL_PROVIDER string - Nhà cung cấp model mặc định

Cài đặt proxy

Tham số Loại Mô tả
PROXY_URL string Proxy URL, hỗ trợ HTTP, HTTPS, SOCKS5
PROXY_ENABLED_PROVIDERS array Danh sách nhà cung cấp dùng proxy

Quản trị dịch vụ

Tham số Loại Mặc định Mô tả
REQUEST_MAX_RETRIES number 3 Số lần thử lại tối đa của nhà cung cấp
REQUEST_BASE_DELAY number 1000 Độ trễ thử lại cơ bản (mili giây)
CREDENTIAL_SWITCH_MAX_RETRIES number 5 Số lần thử tối đa khi chuyển xác thực lỗi
RATE_LIMIT_COOLDOWN_ENABLED boolean false Có tạm thời cooldown tài khoản sau 429 không
RATE_LIMIT_COOLDOWN_MS number 30000 Thời lượng cooldown 429 mặc định (mili giây)
MAX_ERROR_COUNT number 10 Số lỗi tối đa của nhà cung cấp trước khi đánh dấu không khỏe mạnh
WARMUP_TARGET number 0 Số node tự làm mới khi khởi động
REFRESH_CONCURRENCY_PER_PROVIDER number 1 Số tác vụ làm mới song song tối đa mỗi nhà cung cấp

Cài đặt nhật ký

Tham số Loại Mô tả
LOG_ENABLED boolean Bật ghi nhật ký
LOG_OUTPUT_MODE string Chế độ xuất nhật ký
PROMPT_LOG_MODE string Chế độ nhật ký: none, console hoặc file

Ví dụ cấu hình

{
    "REQUIRED_API_KEY": "your-api-key",
    "SERVER_PORT": 3000,
    "HOST": "0.0.0.0",
    "MODEL_PROVIDER": "gemini-cli-oauth,claude-kiro-oauth",
    "PROXY_URL": "http://127.0.0.1:7890",
    "PROXY_ENABLED_PROVIDERS": ["gemini-cli-oauth", "claude-kiro-oauth"],
    "REQUEST_MAX_RETRIES": 3,
    "MAX_ERROR_COUNT": 10,
    "WARMUP_TARGET": 5,
    "LOG_ENABLED": true,
    "LOG_OUTPUT_MODE": "all"
}

Cấu hình Provider Pool (provider_pools.json)

Provider pool cấu hình nhiều tài khoản để cân bằng tải và chuyển dự phòng. Mỗi loại nhà cung cấp có thể có nhiều node tài khoản.

Tham số cấu hình node

Tham số Loại Mô tả
uuid string Định danh node duy nhất, tự động tạo
name string Tên node tùy chỉnh
oauthCredsFilePath string Đường dẫn file xác thực OAuth
checkHealth boolean Bật kiểm tra sức khỏe
checkModel string Model dùng cho kiểm tra sức khỏe
notSupportedModels array Danh sách model không hỗ trợ cho node này
disabled boolean Có tắt node này không

Ví dụ cấu hình

{
    "gemini-cli-oauth": [
        {
            "uuid": "gemini-account-1",
            "name": "Tài khoản Gemini 1",
            "oauthCredsFilePath": "configs/gemini/oauth_creds_1.json",
            "checkHealth": true,
            "checkModel": "gemini-3-flash-preview"
        }
    ],
    "claude-kiro-oauth": [
        {
            "uuid": "kiro-account-1",
            "name": "Tài khoản Kiro 1",
            "oauthCredsFilePath": "configs/kiro/kiro-auth-token.json",
            "checkHealth": true
        }
    ]
}

Cấu hình Fallback

Khi tất cả tài khoản của một loại nhà cung cấp không khả dụng, tự động chuyển sang nhà cung cấp dự phòng đã cấu hình.

Chuỗi Fallback đa loại

Cấu hình providerFallbackChain trong config.json để chỉ định loại dự phòng cho mỗi nhà cung cấp:

{
    "providerFallbackChain": {
        "gemini-cli-oauth": ["gemini-antigravity"],
        "gemini-antigravity": ["gemini-cli-oauth"],
        "claude-kiro-oauth": ["claude-custom"],
        "claude-custom": ["claude-kiro-oauth"]
    }
}

Ánh xạ model đa giao thức

Khi nhà cung cấp chính không khả dụng, ánh xạ model cụ thể sang nhà cung cấp giao thức khác:

{
    "modelFallbackMapping": {
        "gemini-claude-opus-4-5-thinking": {
            "targetProviderType": "claude-kiro-oauth",
            "targetModel": "claude-opus-4-5"
        }
    }
}
