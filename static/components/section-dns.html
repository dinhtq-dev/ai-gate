DNS Redirection

Bật để chuyển traffic công cụ qua MITM (ghi vào file hosts). Tắt để khôi phục routing bình thường. Độc lập với Start/Stop MITM.

DNS Redirection

Trên Windows có thể hiện UAC khi bật/tắt DNS — cần phê duyệt để sửa file hosts. Trạng thái phản ánh file hosts thực tế, không phải cache trình duyệt.
