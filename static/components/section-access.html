Truy cập nhanh

Giữ API Key công khai, base URL và các route nhà cung cấp ở một nơi để sao chép trực tiếp vào client.

-- Key công khai
0 Nhà cung cấp tải sẵn
0 Nhóm nhà cung cấp
0 Tổng số node

Thông tin kết nối

Đây là key mà client phải gửi khi gọi `/v1/chat/completions` hoặc `/v1/messages`.
Thông thường bạn có thể nối thêm route nhà cung cấp vào base URL này và gửi trực tiếp cho client.

Hướng dẫn nhanh

Phù hợp cho lần bàn giao client đầu tiên: lấy key, chọn route, rồi sao chép đoạn cấu hình cuối cùng.

1
Sao chép key trước

Chia sẻ key công khai với client và gửi trong header `Authorization: Bearer KEY`.

2
Chọn route nhà cung cấp

Mỗi nhà cung cấp dùng đường dẫn khác nhau, ví dụ `openai-codex-oauth` hoặc `gemini-cli-oauth`.

3
Sao chép endpoint cuối cùng

Mỗi hàng bên dưới cung cấp endpoint giao thức OpenAI và Claude, sẵn sàng sao chép.

Nếu chưa có node nào được cấu hình, hãy hoàn tất thiết lập trong Cấu hình và Nhóm nhà cung cấp trước, rồi quay lại sao chép đoạn cấu hình.
